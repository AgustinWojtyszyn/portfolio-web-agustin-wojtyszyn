# Portfolio Web — Agustín Wojtyszyn

![Portfolio Screenshot](src/images/profilepic.jpg)

## Desarrollador Web Full Stack

Portfolio profesional desarrollado con **Next.js**, **TypeScript** y **Tailwind CSS**, diseñado para presentar proyectos, experiencia y habilidades en desarrollo web.

El sitio incluye animaciones modernas, fondo espacial interactivo, secciones dinámicas y diseño responsive optimizado para todos los dispositivos.
